## 📋 Project Overview

This project demonstrates a complete DevOps lifecycle for a static HTML/CSS university website. It covers:

- Static website with 5 pages (HTML5 + CSS3)
- Docker containerization with Nginx
- Git Flow branching strategy
- 3 separate CI/CD pipelines (development, staging, production)
- Deployment to Render.com via GitHub Actions
- GitHub Environments with secrets management
